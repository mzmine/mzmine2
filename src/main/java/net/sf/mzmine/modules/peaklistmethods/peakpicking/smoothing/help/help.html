Smoothing

Description

This module applies Savitzky-Golay smoothing to the peak chromatograms in a peak list. Its main purpose is to remove high frequency noise from the chromatograms prior to deconvolution.

Method parameters

Filename suffix
The text to append to the name of the smoothed peak list.
Filter width
The number of data points over which the filter kernel extends. The wider the filter the greater the smoothing effect.
Remove original peak list
Whether to remove the original peak list once smoothing is complete.
