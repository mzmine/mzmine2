Extract scans - Spectra/mass list to csv export

Description

This module exports scans or mass lists to csv text files.

Method parameters

Optional mass list:
If checked, mass lists are exported instead of raw scans
Raw data files:
One or more raw data files that contain the scans/mass lists
Use center time:
Specifies that the center time and scan count should be used (inactive in the case that auto search max is selected)
Scan count to be exported:
Only used for center time and for auto search max
Center time:
Used if option above is checked and auto max is disabled
Time range:
Time range to export scans. Only used if use center time and auto search max is unchecked
Auto search max:
Searches for TIC maximum to center the scan export of the specified count of scans. (Important: Disables the center time and time range)
Export header:
Export meta data as a header for each scan
