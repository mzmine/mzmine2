MS/MS visualizer

Description

This tool displays a two-dimensional plot of the data in a Data Dependent Acquisition (DDA) or Information Dependent Acquisition (IDA) experiment. The x-axis corresponds to retention time of the ms/ms spectrum and the y-axis is the m/z value of the precursor ion of the ms/ms spectrum.

This tool is very useful to get an overview of large amounts of MS/MS data.

Parameters

Raw data file
List of raw data file to display in the MS/MS visualizer. Only one file can be selected.
Retention time
Retention time range.
m/z range
Range of m/z values for precursor ions in ms/ms scans.
Intensity
The intensity of the data points can be calculated based on either:
- Total intensity of the MS/MS scan (=summing of intensity from all peaks)
- Intensity of the precursor ion in the MS scan
Normalize by
The color of individual data points can be normalized based on either:
- All data points in the MS/MS plot
- Data points with a m/z within 10ppm
Min. MS/MS peak intensity
The minimum intensity of a single MS/MS ion which has to be present in the MS/MS spectrum for it to be included in the MS/MS plot.
Set to 0 to show all data points.

Functionality

Double clicking on any data point will display the corresponding MS/MS spectrum.

The following zoom functions are available:

  • Scroll with the mouse to zoom in or out.
  • Drag the mouse from left to right to select the area to zoom in.
  • Drag the mouse from right to left to zoom out to the default view.
  • Single click on the y-axis to auto set the intensity to auto height.
  • Double click on the y- or x-axis to reset the zoom to default.

At the bottom of the plot, it is possible to add peaks from a peak list to the plot. These will show up as green lines. Please make sure that a peak list is available for the raw data file otherwise the dropdown box will be empty.

At the right of the plot, the second button allows to toggle displaying of the data points for the peaks. And the third button allows to toggle displaying of tool tips on the peaks.

The last button to the right of the plot provides search functionality for specific fragment ions in the MS/MS spectra. After entering the m/z of the ion, ppm tolerance, min intensity, if you are looking for a neutral loss ion and color in the dialog prompt, all data points which fulfil the criteria will be colored with the selected color and with the shade based on the normalization.
