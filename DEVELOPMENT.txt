
To start MZmine development, please follow the tutorial available at MZmine website.
Make sure Apache Maven is installed.
 
To run the automatic jUnit tests, run the following command:

mvn test
 
To build the MZmine package from the sources, run the following command:

mvn clean package

The final MZmine distribution will be placed in target/MZmine-2.**.zip

If you encounter any problems, please contact the developers:
https://groups.google.com/forum/#!forum/mzmine-devel
